## Installation

Install my-project with yarn

```bash
  yarn install
  cd mern-chat
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/imHardik1606/mern-chat
```

Go to the project directory

```bash
  cd mern-chat
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  nodemon index.js
```

Start the vite

```bash
  yarn dev
```
